# **First-Project**
This is my first Unreal Engine 5 project. It introduced me to the basics of game development and sparked my excitement about the engine’s endless possibilities.

The player can control the default UE5 character and explore a custom-designed level. The level features a simple puzzle, solving it turns ON the portal that enables Player to complete the game 🏆.

## 🧠 What I learned
* Navigating around the engine
* Placing meshes onto the scene
* Using shortcuts W/E/R to toggle moving/rotating/scaling
* How to manage assets in folders to not get them cluttered (for future projects 😅)
* Using custom postprocessing volumes
