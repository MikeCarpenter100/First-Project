# **First-Project**
This is my first Unreal Engine 5 project. It introduced me to the basics of game development and sparked my excitement about the engine’s endless possibilities.

The player can control the default UE5 character and explore a custom-designed level. The level features a simple puzzle, and solving it completes the game 🏆.
